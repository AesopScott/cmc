# CMC Website

cmcenters.org — converts JSON data extracts into HTML/JS pages. Astro-based static site.

## Stack
- Framework: Astro
- Data: JSON source files in `data/`
- Output: Generated HTML in `dist/`
- Build: `npm run build`

## Key files
- `src/` — build scripts and templates
- `data/` — JSON source files
- `dist/` — generated HTML output
- `astro.config.mjs` — Astro config
